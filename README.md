# ruwaqmaqui

Project template with **TanStack Router** + **Bun** + **SQLite** (Drizzle ORM) + **Better Auth** + **Hono** + **Docker**.

## Stack

| Layer | Tool |
|-------|------|
| Runtime | Bun 1.3 |
| Frontend | React 19, TanStack Router, Vite 8 |
| Backend | Hono 4 (Bun server) |
| Auth | Better Auth (email/password) |
| Database | SQLite via `bun:sqlite` + Drizzle ORM |
| Linter | Oxlint |
| Formatter | Oxfmt |
| Container | Docker + Compose |

## Scripts

```bash
bun run dev            # client + server (dev)
bun run dev:client     # Vite dev server only
bun run dev:server     # Bun server with --watch
bun run build          # build client
bun run start          # production server
bun run lint           # Oxlint
bun run format         # Oxfmt check
bun run format:fix     # Oxfmt write
bun run typecheck      # tsc --noEmit
bun run db:push        # push schema to SQLite
bun run db:studio      # Drizzle Studio (web UI)
bun run db:generate    # generate migration files
bun run gen:routes     # regenerate TanStack route tree
```

## ENV

```env
DATABASE_URL=./data/sqlite.db
PORT=3000
NODE_ENV=development
```

## Docker

```bash
docker compose up --build
```

## Project structure

```
src/
├── client/          → React app
├── routes/          → TanStack file-based routes
├── server/
│   ├── index.ts     → Hono entry point
│   ├── auth.ts      → Better Auth config
│   └── db/
│       ├── index.ts → DB connection (bun:sqlite)
│       └── schema.ts → Drizzle schema
├── routeTree.gen.ts → auto-generated by tsr
└── vite-env.d.ts
```
